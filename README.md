# Space Engine (Gravity-Free Cosmic Simulator)

ニュートンの万有引力やダークマターといった「見えない引力（質量＝引力）」の前提を捨て去り、宇宙空間を「流体」として扱うことで、星の形成や銀河の大規模構造を再現するGPUベースの物理シミュレーターです。

## 物理モデルのコンセプト

- **重力の正体は圧力勾配**: 物質が互いに引っ張り合うのではなく、空間の流体力学的な「圧力勾配」によって物質が動く。
- **渦による負圧**: 空間の速度場（風）が渦を巻くことで中心に負圧が発生し、そこへ周囲の星やガスが引き寄せられる（ベルヌーイの定理）。
- **自然な多様性とスイングバイ**: 星は乱数ではなく、自身が飛び込んだ先の空間の渦との「速度差」によって、素直に渦に従うか、弾き出されて「はぐれ星」になるかが決まる。

## 必要なライブラリ

- NumPy
- CuPy (GPU計算用)
- VisPy (高速描画用)
- glfw (描画バックエンド)

## 実行方法

```bash
python simlator.py
```
